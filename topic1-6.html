1.6 Research and Document an Ergonomic Issue in HCI

Cognitive ergonomics is the analysis and design of human mental processes and the impacts they have on the human brain when interacting with a given system. An example of utilizing cognitive ergonomics is designing an interface that is easy to use based on data gathered from measuring the cognitive load of using the interface. An intuitively designed user interface is going to have less cognitive impact then that a poorly designed user interface. Instead of dealing with traditional ergonomic issues like carpel tunnel or repetitive strain injury, cognitive ergonomics deals with the mental impacts, like stress, fatigue and human reliability.

Cognitive ergonomics sets out to measure the cognitive demands and skills required to complete different tasks. Special frameworks like GOMS are utilized to measure these cognitive strains against a special set of models. There a quite a few models that all follow the same general set of principles, but are all specialized for their intended field. These tools can also measure the users performance while completing human-computer interactions.

Human computer interaction leverages cognitive ergonomics when designing and implementing user interfaces, applications, databases. It is also applied in the fields of aviation, medical system design and transportation.

Whenever we have an excellent experience with an application, or think to ourselves, “Wow that was easy to use”, you can be sure that the company behind that product was heavily invested into cognitive ergonomics.
