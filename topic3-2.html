3.2 Customize and Adapt the Windows Interface

Windows 10 has a wide variety of customization options that a lot of people are unaware of. I will go through where to find and how to change some of the more common settings.

One of the easiest things a user can do to change their GUI is changing the desktop background. You can do this by right clicking anywhere on the empty part of the desktop and selecting the Personalize option. Once in the personalization options, you can click the Browse button and select any image on your computer to be your desktop background. In the left hand navigate pane of the Personalization menu there is a Themes tab which has options for changing the default Windows theme, downloading more custom themes, changing Windows default sounds and even the mouse pointer. There is also a Fonts tab for changing the default system fonts and a Start tab which has options for changing the appearance and functionality of the Start menu. I was shocked to see an option to re-enable the old Windows 8.0 style full screen start menu, I would like to see the statistics on how many users actually utilize this.

To access the Mouse and Keyboard options, utilize the Windows Search functionality by typing mouse into the search bar. Select the Mouse settings option and from there you can configure cursor speed and if you are left-handed you can change the primary button here as well. For laptop users there is a Touchpad tab on the left-hand side of the Devices pane. From here you can disable the touch pad entirely, change the cursor speed and sensitivity, and change any of the touchpad figure gestures.

Windows accessibility options are found by typing and selecting settings in the Windows Search bar. Ease of Access is one of the options under the Windows Settings pane. Windows has quite a few accessibility options to choose from. The Display tab has options for changing font and icon sizes, brightness settings, and enabling/disabling different Windows effects. There is a Magnifier tool which lets the user keep their default screen resolution size and zoom in/out on the fly. There is a tab for Color filters which makes the operating system usable for those who are colour blind. Another interesting feature is the Narrator which is a built-in screen reader to aid those who have vision impairments. There is even built in speech recognition software right in the operating system.

Windows 10 is a highly customizable and accessible modern operating system. I only covered some of the built-in configuration options that are available. There are countless plugins and applications that can customize the appearance, change functionality and add accessibility.
