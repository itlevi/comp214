Specialized Input Devices and Their Applications

The 3D mouse is a specialized input device used to make navigating in virtual 3D environments easier. It is occasionaly used in architecture, CAD, 3D design and 3D modelling.

https://www.youtube.com/watch?v=fe7_ooiA170

A graphics tablet is a specialized input device used to make drawing on a computer more effective, easier to use and provide a more natural drawing experience. They are essential for any artist to make quality drawings on a computer. The mechanics of how the tablet functions different from tablet to tablet, but they all map the location of the pen in relation to a location on the users monitor.
