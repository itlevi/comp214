Emerging Interface Concepts and Technologies

The future of interfaces is going to be moving away from the once king, keyboard and mouse combo. Emerging technologies in voice recognition, virtual reality and augmented reality have begun to change the HCI paradigm. With mobile devices becoming more powerful and more capable people are becoming less reliant on the immobile desktop PC.

Hongkiat has listed the following 8 key features that future human-computer interfaces are going to use:

  1. 1. Gesture Interfaces

    Hand, face and body gestures to perform tasks.

  2. 2. Brain-Computer Interface

    Device that can translate signals from the brain and use them as computer inputs.

  3. 3. Flexible OLED display

    The next generation of touch screens.

  4. 4. Augmented Reality (AR)

    Utilizes the natural environment to project interfaces on any surface or object.

  5. 5. Voice User Interface (VUI)

    Takes the users voice and translates it to different commands.

  6. 6. Tangible User Interface (TUI)

    Interacts with real world objects to create an immersive user experience.

  7. 7. Wearable Computer

    Various devices like smart watches, augmented reality glasses and other smart accessories.

  8. 8. Sensor Network User Interface (SNUI)

    Multiple devices that can be joined together to complete a single interface.
